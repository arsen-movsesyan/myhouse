Development draft
Basic logic for user administration
  • Any user may belong to one household only
  • User can assign himself to administrator only while initiating household
  • Household may have many estates
  • After registration, user must provide personal information and choose to be administrator or to join existing household
  • In order to join existing household, user must create join request (Draft)
  • Starting new household, user assigns himself as administrator
  • Administrator cannot join existing household
  • Administrator may grant access to user to join his household
  • Household may have multiple administrators (Draft)
